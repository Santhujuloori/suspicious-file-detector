## 📌 About the Project

The **Suspicious File Detector** is a cybersecurity web application that helps identify potentially malicious or risky files based on their characteristics.

This tool allows users to upload files or analyze file details and flags suspicious patterns such as unusual extensions, file names, or known threat indicators.

---

## 🎯 Key Objectives

* Assist SOC analysts and beginners in identifying suspicious files
* Detect risky file types and patterns
* Provide quick and simple file analysis

---

## ⚙️ Features

* 📁 Upload files for analysis
* 🚨 Detect suspicious file extensions (e.g., .exe, .bat, .js)
* 🔍 Identify abnormal file naming patterns
* ⚡ Fast and easy-to-use interface
* 📊 Basic insights for security awareness
